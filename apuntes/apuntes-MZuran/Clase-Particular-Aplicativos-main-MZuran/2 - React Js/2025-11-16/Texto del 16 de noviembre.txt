Definición de Funciones:

	duplicar

Ejecución de Funciones:

Directorios importantes dentro de nuestro proyecto de react:

src/				-> Aquí adentro vamos a tener TODO EL CÓDIGO QUE VAYAMOS A ESCRIBIR
src/hooks/			-> Acá vamos a tener todos los "hooks" personalizados que les va a dar su profesor
src/compontents/		-> Acá adentro vamos a tener todos los "componentes" de react que vayamos creando

src/page.js			-> Va a ser el "home" de nuestra página
src/layout.js			-> Es un archivo que contiene componentes que se comparten con todas las otras páginas (subcarpetas)

src/app/<nombre>/page.js	-> Van a ser los distintos directorios de nuestra página. Por ejemplo /login
src/app/<nombre>/layout.js	-> Es un layout que tiene los componentes compartidos de el page actual y todas las subcarpetas

node_modules/			-> Es la carpeta que contiene todos los paquetes. Se crea con npm i o npm install.
public/				-> Es la carpeta que contiene los archivos multimedia (fotos, videos, fuentes tipográficas, etc)
package.json			-> Es el archivo que contiene los detalles técnicos del proyecto

Una dependencia es un paquete de node que el proyecto necesita para correr
