Bases de Datos Relaciones
Las Bases de Datos SQL pueden ser de de tipo relacionales o no relacionales. En el caso de las relacionales se estructura la información en tablas, filas y columnas. Las mismas tienen la capacidad de establecer relaciones de la información entre distintas tablas por medio de claves foráneas.
También existen bases de datos no relacionales que no tienen porque estructurar su información en tablas, filas y columnas.

Creación de tablas
Una vez, creada la tabla Grades, podemos crear la tabla Students que tiene una clave foránea que referencia al id de Grades.
Los tipos de datos que estoy utilizando son:
int para los números,
varchar(255) para el texto (con un máximo de 255 caracteres)
La clave foránea primero tiene que ser creada como una columna más con el mismo tipo de dato que en la tabla a la que se va a referenciar.
