Temas de la Clase

En esta sesión aprenderemos los siguientes conceptos fundamentales de JavaScript:

  • array.pop() — Elimina y devuelve el último elemento del array.
  • array.push() — Agrega uno o más elementos al final del array.
  • array.map() — Crea un nuevo array aplicando una función a cada elemento.
  • array.forEach() — Ejecuta una función por cada elemento del array sin crear uno nuevo.
  • array.filter() — Devuelve un nuevo array con los elementos que cumplan una condición.
